# Yudo 餐厅中文点餐与外卖订单系统

这是为巴厘岛 Yudo／鱼多多中餐厅制作的移动端优先点餐 MVP。顾客可以浏览中文菜单、选择辣度、加入购物车、填写酒店配送信息，并生成一份可直接发给中文客服确认的订单。

## 已有功能

- 中文菜单、分类筛选与菜名搜索
- 菜品辣度选择
- 购物车增减、删除与金额计算
- 购物车自动保存在当前浏览器
- 外卖配送／到店自取切换
- 联系人、酒店地址、人数及用餐时间收集
- 儿童、老人及免费餐具提醒
- 自动生成并复制标准中文订单
- 手机、平板和电脑响应式布局

## 本地运行

这是一个无依赖的静态网页项目。直接双击 `index.html` 即可预览，也可以在项目目录运行：

```bash
python3 -m http.server 8080
```

然后打开 `http://localhost:8080`。

## 项目结构

```text
.
├── index.html   # 页面结构与结账表单
├── styles.css   # 品牌视觉与响应式布局

```

## 上线前需要补充

当前版本用于验证点餐流程，订单仍需顾客复制后发给客服。正式上线建议继续接入：

1. Yudo 的真实菜品、图片、价格与库存；
2. WhatsApp 或企业客服的一键发送入口；
3. 根据酒店区域自动计算配送费；
4. 后台订单管理、接单状态和骑手进度；
5. 在线支付、退款及改单流程；
6. 印尼语厨房小票。

## 说明

演示菜单和价格仅用于产品原型，请在正式营业使用前由餐厅审核替换。
